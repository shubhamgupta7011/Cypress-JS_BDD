# Automation framework setup

## Pre-requists

- Node is installed (version above 14)
- IDE
- git installed
- Access to github

*use git bash for all the below commands*
## clone the repo


Git clone using ssh
``` bash
git clone git@github.com:inditex/wsc-testingmanagementocp.git
```

## Install dependencies
npm will download node dependencies from inditex npm repo url set in .npmrc
``` bash
npm install
```
